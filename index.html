Privacy Policy

Effective Date: April 15, 2025

This privacy policy applies to the Simple Screenshot Extension (the "Extension").

Data Collection

The Extension does not collect, store, or transmit any personal or sensitive user data.

How the Extension Works

The Extension operates entirely within your browser. When activated by the user, it captures a screenshot of the current web page and allows the user to save it directly to their device. No information is sent to any servers.

Third-Party Services

The Extension does not use any third-party services, analytics tools, or cloud storage. All functionality is self-contained and local to the user's browser.

Permissions

The Extension requests only the minimal permissions required to perform its core function, including access to the current tab and downloads API to save images locally.
